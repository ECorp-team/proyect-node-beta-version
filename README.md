## Documentation

La estructura del FrontEnd debe ser la siguiente:

```bash
/project-directory
│
├── /views
│   ├── index.html
│   └── /src
│       └── /css
│           └── style.css
```

## Installation

Para comenzar a editar el codigo del repo

```bash
  cd Documents
  git clone https://github.com/ECorp-team/proyect-node-beta-version.git
```

## Subir cambios

```bash
  git add .
  git commit -m "comment"
  git push origin main
```
